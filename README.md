# odin-recipes

Welcome to **Odin Recipes**, a simple and easy-to-navigate website dedicated to classic Filipino home cooking. This project showcases a collection of traditional recipes, complete with step-by-step instructions and images, making it beginner-friendly and perfect for anyone wanting to explore Filipino cuisine.

---

## Features

- Categorized recipe sections: Soups, Hearty Stews, Noodles, and Snacks & Appetizers.
- Step-by-step cooking instructions for each dish.
- Images for every recipe to help visualize the finished dish.
- Easy navigation between recipes and back to the homepage.
- All recipes are designed for 2–3 servings, ideal for small families or beginners.

## How to Use

1. Clone or download this repository to your computer.
2. Open the `index.html` file in your preferred web browser.
3. Browse recipes by category and click **View Recipe** to see ingredients and cooking steps.
4. Navigate between recipes easily using the previous/next links at the top and bottom of each page.
